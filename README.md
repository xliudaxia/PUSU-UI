# Pusu-UI

Pure Tiny React UI Library.

## 安装

> 使用npm

~~~sh
npm install pusu-ui --save
~~~

> 使用yarn

~~~sh
yarn add pusu-ui
~~~

## 使用

```js
// 加载样式
import "pusu-ui/dist/index.css";

// 引入组件
import { Button } from 'pusu-ui'
```

### 技术栈

- Typescript
- ……

### 相关命令

```sh
// 启动开发环境
yarn start
```
